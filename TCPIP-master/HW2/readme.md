## How to use:
1.  Compile appServer.c & appClient.c
    
        gcc appServer.c -o Server.out
        gcc appClient.c -o Client.out

2. Run Server.out
    
        ./Server.out [portNumber]
3. Run appClient.out

        ./Client.out [IP addr] [portNumber]
4.  Then you will see the result.
